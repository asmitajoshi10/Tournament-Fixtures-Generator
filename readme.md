# Full-Stack Tournament Scheduler Engine

An automated, full-stack scheduling matrix that leverages algorithmic optimization to eliminate team time-slot collisions across multi-day sporting events.

##  Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (Dynamic DOM manipulation)
- **Backend:** Node.js, Express.js (REST API architecture)
- **Database:** MongoDB Atlas (Cloud Hosted Multi-Shard Replica Set)

##  Key Features
- **Anti-Collision Shield:** Prevents teams and specific sports grounds from being double-booked.
- **Fisher-Yates Randomization:** Dynamically distributes high-intensity sports evenly across the calendar window.
- **Cloud Persistence:** Securely commits generated schedule blueprints to MongoDB Atlas.
